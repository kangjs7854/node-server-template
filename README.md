## node-server-template
> node开发接口的项目模板，主要是用来开发一些api，所以删除了express生成器的一些模板引擎的无用代码，让目录结构更加简洁

1. 安装依赖
```
npm i 

```

2. 启动服务
> 通过nodemon实现了热更新，方便开发
```
npm run start

```
